flutterDesktopз”ЁAndroid Studioзј–иҜ‘иҝҗиЎҢ

еҸ‘еёғдәҺ 2019-05-24 | еҲҶзұ»дәҺ windows гҖҒ AndroidStudio гҖҒ flutter гҖҒ dart гҖҒ еӯҰд№  гҖҒ app | 3еҲҶй’ҹ | 528еӯ—ж•° | жөҸи§ҲйҮҸпјҡ

иҝҷзҜҮж–Үз« ,жҳҜеҹәдәҺWindows 10 1903 зҡ„еҰӮжһңдҪ жҳҜз”Ёзҡ„е…¶д»–зі»з»ҹ,еҸҜиғҪеҪ“еүҚж•ҷзЁӢдёҚйҖӮеҗҲ,еҪ“з„¶д№ҹеҸҜд»ҘеҒҡеҸӮиҖғ

йңҖиҰҒеҮҶеӨҮ:

  • жӯЈеёёзҡ„ windows 10 зі»з»ҹ з”өи„‘дёҖеҸ°
  • жӯЈеёёзҡ„зҪ‘з»ңзҺҜеўғ
  • зЎ¬зӣҳеү©дҪҷз©әй—ҙ > 10G
  • жӯЈеёёзҡ„flutterзҺҜеўғ (flutterзҺҜеўғзҡ„жҗӯе»әзҪ‘дёҠиө„ж–ҷеҫҲеӨҡ,дёҖжүҫдёҖеӨ§зүҮ)
  • Android Studio 3.4.1 AS3.4.1дёӢиҪҪ
  • Visual Studio 2017 жҲ–иҖ… 2019 (жҲ‘з”Ёзҡ„2019зӨҫеҢәзүҲ) VS2019

жҺҘдёӢжқҘ, Visual Studio з®Җз§°дёәVS,гҖҖAndroid Studioз®Җз§°дёәAS

жӯҘе…ҘжӯЈйўҳ:

  • е…ӢйҡҶйЎ№зӣ® flutter-desktop-embedding
    йЎ№зӣ®е…ӢйҡҶи·Ҝеҫ„дёәдҪ зҡ„flutterжүҖеңЁзӣ®еҪ•
  • е®үиЈ…VS,йҖүжӢ©"дҪҝз”ЁC++зҡ„жЎҢйқўејҖеҸ‘" и®°дҪҸе®үиЈ…и·Ҝеҫ„,еҗҺиҫ№иҰҒз”Ё!
  • е®үиЈ…е®ҢжҲҗVSд№ӢеҗҺ,е°Ҷе®үиЈ…и·Ҝеҫ„дёӢзҡ„"\2019\Community\VC\Auxiliary\Build"е®Ңж•ҙи·Ҝеҫ„ж·»еҠ зҺҜеҲ°еўғеҸҳйҮҸpath;

  • AS е®үиЈ…дёҠflutterжҸ’д»¶ flutter i18nжҸ’д»¶ dartжҸ’д»¶,йҮҚеҗҜAS;
  • жҠҠflutterеҲҶж”ҜеҲҮжҚўеҲ°master,еҗҜеҠЁAS,жү“ејҖflutter-desktop-embeddingзӣ®еҪ•дёӯзҡ„exampleйЎ№зӣ®;
  • зӯүйЎ№зӣ®еҠ иҪҪе®ҢжҜ•,жү“ејҖASеә•йғЁзҡ„Terminal,иҝҗиЎҢ(еҰӮжһңжІЎжңүиҝҷдёӘйқўжқҝ,еҸҜд»ҘеңЁview -> Tool Windows -> Terminal жү“ејҖ):
set ENABLE_FLUTTER_DESKTOP=true

з„¶еҗҺиҝҗиЎҢжҹҘзңӢжҳҜеҗҰи®ҫзҪ®жҲҗеҠҹ,

flutter devices

еҰӮжһңжҲҗеҠҹзҡ„иҜқ,дјҡжңүи®ҫеӨҮеҲ—иЎЁ;
жҺҘдёӢжқҘе°ұжҳҜиҝҗиЎҢйЎ№зӣ®дәҶ

flutter run

еӨҡи®ҫеӨҮзҡ„жғ…еҶөдёӢ,дјҡжҸҗйҶ’еӨҡдёӘи®ҫеӨҮ.жҢҮе®ҡи®ҫеӨҮеҚіеҸҜ:

flutter run -d windows

жҲ‘жүҖйҒҮеҲ°зҡ„й”ҷиҜҜ д»ҘеҸҠжҺ’йҷӨж–№жЎҲ:

  • иҝҗиЎҢжҠҘй”ҷ: unable to build: could not find vcvars64.bat
    жЈҖжҹҘзҺҜеўғеҸҳйҮҸpathдёӯ,vsдёӯbuildзӣ®еҪ•дёӯvcvars64.bat ж–Үд»¶жүҖеңЁзҡ„и·Ҝеҫ„жҳҜеҗҰжӯЈзЎ®.и·Ҝеҫ„йңҖиҰҒе®Ңж•ҙи·Ҝеҫ„жҜ”еҰӮжҲ‘зҡ„,й»ҳи®Өи·Ҝеҫ„:"C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\VC\Auxiliary\Build"
  • иҝҗиЎҢжҠҘй”ҷ:
Build process failed
'xxx\windows\build.bat' is not recognized as an internal or external command,
operable program or batch file.
